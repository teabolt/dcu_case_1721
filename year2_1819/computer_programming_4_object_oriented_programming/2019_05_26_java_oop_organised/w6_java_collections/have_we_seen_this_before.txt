Title: Have we seen this before
Description: Write a program which reads numbers until a -1 is encountered and only print out numbers which have previously occurred in the input. You should use a Set to aid you in solving this problem.
E.g.
java HasPrevious
Enter some numbers (-1 to end)
45 8 20 7 16 45 0 22 20 6 20 9 -1
Previous:
45
20
20

Note that the 45 occurs twice and the 20 occurs three times and the numbers are only printed of they haven't occurred previously. 
Tests: 2 tests run/ 2 tests passed
Grade: 100
